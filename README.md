# Kevin_bot

Just a Kevin bot.

## Installation (Windows)

此腳本依賴下列項目，必須事前安裝

- **[Python 3.6 above](https://www.python.org/downloads/)**

安裝完 Python 後在 console 以下指令獲取 discord lib：
```console
pip install discord.py
```

## Getting started 

**記得先將 setting.json 的 TOKEN 改成自己機器人的 token 後輸入下列指令運行機器人：**
```console
python bot.py
```
